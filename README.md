## Descrição

Base de desenvolvimento backend de SMUL/ATIC:

- NESTJS: https://docs.nestjs.com/
- PRISMAIO: https://www.prisma.io/docs/getting-started

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

Por padrão, a aplicação rodará na porta 3000.

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run prod
```
